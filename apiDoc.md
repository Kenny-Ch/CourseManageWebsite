### 登录系统并检测登录是否异常

#### 请求方式

~~~
http请求方式: POST
http://localhost:8080/user/lgoin
~~~

#### 请求参数

|   参数   | 是否必填 |   说明   |
| :------: | :------: | :------: |
|  email   |    是    | 邮箱地址 |
| password |    是    |   密码   |

#### 返回格式

JSON

~~~
{
       code:401,
       msg:'password required'
}
~~~

#### 返回参数

| 代码 |               说明文字                |
| :--: | :-----------------------------------: |
| 401  |              未输入密码               |
| 402  |              未输入email              |
| 403  |             账号密码错误              |
| 200  | 登录成功（会将个人信息保存至session） |
